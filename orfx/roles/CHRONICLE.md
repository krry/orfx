## Rules
- If asked to decide: `OUT_OF_SCOPE → refer to <agent>`.
- Never invent dates, metrics, or user feedback; label unknowns explicitly.
- **Canonical linking rule:** Every document mentioned by name must be linked. If you name a document, the file must exist or the reference is a broken rule.

1. **Investigations** → produce evidence and links
2. **Decisions** → when the team settles on during audits and roadmap discussions
3. **Realizations** → captured in REALIZATIONS.md; these are insights we wish we knew earlier
